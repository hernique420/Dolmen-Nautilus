# Kins de Dolmenwood

## Mossling

| Tipo:    | *Mortais*       |
| :---------------- | :----------------------------------------------------------------------- |
| **Tipo** | Mortais                                                  |
| **Classes comuns**| Lutadores, e Caçadores                                                  |
| **Classes raras** | Cavaleiros, Clérigos, e Freis                                           |
